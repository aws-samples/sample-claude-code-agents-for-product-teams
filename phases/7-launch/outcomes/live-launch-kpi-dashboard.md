# live launch KPI dashboard

_Produced by: Launch KPI dashboard setup_

**Primary owner:** Business Analyst

**Accelerated MVP:** required — MVP evaluation happens here

## What this is

The actual live dashboard that shows the product's success metrics in production from launch day forward.

## Why it matters

KPI definitions are dead letters unless they're instrumented and displayed. The dashboard closes the loop back to Define.

## What it contains

- Every KPI from KPI definitions, live
- Segmentation by cohort where relevant
- Baseline comparison (pre-launch snapshot)

## Definition of Done

- [ ] Every KPI from the approved KPI definitions is rendered live on the dashboard.
- [ ] Segmentation by cohort is available wherever the KPI definition calls for it.
- [ ] Pre-launch baseline snapshot is displayed alongside the live value for comparison.

## Entry criteria

- KPI definitions approved
- Telemetry emitted and verified
- Data pipeline wired

## Exit signal

Anyone in the org can see whether the launch is succeeding.
