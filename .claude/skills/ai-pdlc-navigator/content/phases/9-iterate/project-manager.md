# Project Manager in Iterate

Iterate is where delivery discipline turns into institutional memory. You facilitate lessons-learned, close the project cleanly, run the retrospective with the Sponsor, and keep the benefits-review cadence alive so Iterate decisions don't evaporate once the team moves on.

## What you do

- Facilitate lessons-learned capture so the organization actually learns from this cycle.
- Close the project and archive artifacts so institutional memory is preserved.
- Coordinate benefits realization reviews on the scheduled cadence.
- Run the post-launch retrospective with the Product Sponsor.
- Hand team and capability learnings into the next cycle's resourcing and planning.

## Artifacts you own

- [lessons-learned repository](artifacts/lessons-learned-repository.md) — the captured learnings on record.
- [closed project & archive](artifacts/closed-project-archive.md) — the clean project closure and archive.
- [scheduled benefits reviews](artifacts/scheduled-benefits-reviews.md) — the benefits-review cadence that keeps Iterate honest.
- [retrospective report](artifacts/retrospective-report.md) — the post-launch retrospective, with Product Sponsor.

## Artifacts you contribute to

You don't contribute to others' artifacts this phase.

## Outcomes you drive

You don't drive outcomes this phase — you input into others'.

## Who you work with

- **Product Sponsor** — bi-directional on retrospective and closure posture.
- **Product Owner** — bi-directional on lessons that feed the updated roadmap.
- **Architect** — bi-directional on technical lessons and modernization planning.
- **All role leads** — bi-directional on role-specific lessons-learned contributions.

## Handoff into [the next Discover cycle](../1-discover/README.md)

You exit Iterate with an archived cycle and captured lessons. Carry that into the next Discover cycle's initial project brief and sponsor/PO alignment facilitation — predictable delivery is how each cycle gets easier, and closure discipline is how each team starts the next cycle with a sharper plan than the last.
